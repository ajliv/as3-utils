Classpublic class MaskImage
InheritanceMaskImage flash.display.Sprite

The MaskImage class is a simple solution for masking a self-loaded image.

Loads an image from the URLRequest passed in the constructor as a Bitmap and applies a mask of the passed dimensions (default: 100x100).

The image can be set to scale to fit or fill (depending on the set MaskImageScaleMode). By default no scaling is applied.

The image can be set to be centered within the mask. (depending on the set MaskImageAlignMode). By default the image is aligned to the top-left corner of the mask.

public function MaskImage(urlRequest:URLRequest, maskWidth:Number = 100, maskHeight:Number = 100, scaleMode:String = null, alignMode:String = null)

Creates a new MaskImage instance.

Parameters
urlRequest:URLRequest — The URLRequest for the image to load.
 
maskWidth:Number (default = 100) — The width of the mask.
 
maskHeight:Number (default = 100) — The height of the mask.
 
scaleMode:String (default = null) — The way the image is displayed, can accept any constnt value of the MaskImageScaleMode class. By default is MaskImageAlignMode.NO_SCALE.
 
alignMode:String (default = null) — The way the image is aligned within the box, can accept any constant value of the MaskImageAlignMode class. By default is MaskImageAlignMode.TOP_LEFT.

Events
Event.COMPLETE — by the MaskImage when the image is added to the display list after loading.
