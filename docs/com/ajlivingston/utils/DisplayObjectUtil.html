scaleToFit()method 
public static function scaleToFit(obj:DisplayObject, contain:DisplayObject = null, center:Boolean = true, addTo:Boolean = false, addToIndex:int = -1):void

Scales a DisplayObject as large as possible while fitting completely within another DisplayObjct.

Parameters
obj:DisplayObject — The DisplayObject to be scaled.
 
contain:DisplayObject (default = null) — The DisplayObject to scale within. If set to null(default) then obj's parent will be used.
 
center:Boolean (default = true) — Whether or not the scaled DisplayObject is to be centered within the container DisplayObject. Default is true.
 
addTo:Boolean (default = false) — If true, add obj as a child of contain at the addToIndex parameter's value. The contain parameter must be passed a DisplayObjectContainer.
 
addToIndex:int (default = -1) — The index to add obj as a child at in contain if addTo is passed as true. The obj will be added to the top if less than 0.

See also

centerOn()
